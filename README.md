# 歌詞ハイ！タイマー v7 UI

音源を再生しながら、歌詞が来た瞬間に **ハイ！** を押して LRC / SRT / CSV を作るオフラインPWAです。

## v7の変更点

- 操作UIを画面下に固定
- 再生 / 停止ボタンを **ハイ！** ボタンのすぐ近くに配置
- 小さい **消** ボタンを追加
  - 押した時刻に空のLRC行を追加します
  - 例：`[01:56.00]`
  - 間奏で歌詞を消したい時に使います
  - **消** を押しても、次の歌詞には進みません
- 前 / 今押す / 次 / 次の次 の歌詞カンペを固定UI内に表示
- 「表示中」プレビューを追加
  - 現在の再生位置で実際に出ている歌詞を確認できます
  - 空行タイム後は「消えています」と表示します
- 戻す系ボタンは削除
  - ミスした場合は最初から録り直す運用を想定

## キーボード操作

- Space：ハイ！
- Enter：再生 / 停止
- C：消
- R：最初から
- 1 / 2 / 3：0.5倍 / 0.75倍 / 1.0倍

## 使い方

1. 音源を読み込む
2. 歌詞を1行ずつ貼る
3. 「歌詞をセット」
4. 曲を再生
5. 歌い出しに合わせて **ハイ！**
6. 間奏などで字幕を消したいところは **消**
7. LRC出力

音源や歌詞はサーバーに送信されません。端末内だけで動きます。

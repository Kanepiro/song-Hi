# 歌詞ハイ！タイマー v10 UI

音源を再生しながら、歌詞が来た瞬間に **ハイ！** を押して LRC / SRT / CSV を作るオフラインPWAです。

## v10の変更点

- 下部固定UIの並びを変更しました
  - 表示中プレビュー
  - 前 / 今押す / 次 / 次の次 の縦並びカンペ
  - 再生 / 停止 / ハイ！ / 消 の操作ボタン
- **歌詞カンペをハイ！ボタンの上に配置**しました
  - タップ中の指で歌詞が隠れにくくなります
- **次の次** は残しています
- 操作ボタンの下に余白を追加しました
  - ハイ！ボタンがスマホ画面の下端に寄りすぎないようにしています
- サービスワーカーのキャッシュ名を更新しました
  - PWAとして開いた時も新しいUIが反映されやすくなります

## 引き続き入っている機能

- 再生 / 停止ボタンを **ハイ！** ボタンのすぐ近くに配置
- 小さい **消** ボタン
  - 押した時刻に空のLRC行を追加します
  - 例：`[01:56.00]`
  - 間奏で歌詞を消したい時に使います
  - **消** を押しても、次の歌詞には進みません
- 「表示中」プレビュー
  - 現在の再生位置で実際に出ている歌詞を確認できます
  - 空行タイム後は「消えています」と表示します
- 戻す系ボタンなし
  - ミスした場合は最初から録り直す運用を想定

## キーボード操作

- Space：ハイ！
- Enter：再生 / 停止
- C：消
- R：最初から
- 1 / 2 / 3：0.5倍 / 0.75倍 / 1.0倍

## 使い方

1. 音源を読み込む
2. 歌詞を1行ずつ貼る
3. 「歌詞をセット」
4. 曲を再生
5. 歌い出しに合わせて **ハイ！**
6. 間奏などで字幕を消したいところは **消**
7. LRC出力

音源や歌詞はサーバーに送信されません。端末内だけで動きます。

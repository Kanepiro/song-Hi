# 歌詞ハイ！タイマー v12 UI

音源を再生しながら、歌詞が来た瞬間に **ハイ！** を押して LRC / SRT / CSV を作るオフラインPWAです。

## v12の変更点

- 固定オーバーレイを下の画面と見分けやすくしました
  - 背景色を少し変えました
  - 枠線を強めました
  - 影と内側の縁取りを追加しました
- ボタン配置を2行に変更しました
  - 1行目：**最初から / ▶ / 消**
  - 2行目：**ハイ！**
- **ハイ！** は大きい1行ボタンとして残しました
- 歌詞カンペは今まで通り **ハイ！ボタンの上** に表示します
- **前 / 今押す / 次 / 次の次** はすべて残しています
- ハイ！ボタンの下の親指用余白も残しています
- サービスワーカーのキャッシュ名を更新しました

## 引き続き入っている機能

- 小さい **消** ボタン
  - 押した時刻に空のLRC行を追加します
  - 例：`[01:56.00]`
  - 間奏で歌詞を消したい時に使います
  - **消** を押しても、次の歌詞には進みません
- 「表示中」プレビュー
  - 現在の再生位置で実際に出ている歌詞を確認できます
  - 空行タイム後は「消えています」と表示します
- 戻す系ボタンなし
  - ミスした場合は最初から録り直す運用を想定

## キーボード操作

- Space：ハイ！
- Enter：再生 / 停止
- C：消
- R：最初から
- 1 / 2 / 3：0.5倍 / 0.75倍 / 1.0倍

## 使い方

1. 音源を読み込む
2. 歌詞を1行ずつ貼る
3. 「歌詞をセット」
4. 曲を再生
5. 歌い出しに合わせて **ハイ！**
6. 間奏などで字幕を消したいところは **消**
7. LRC出力

音源や歌詞はサーバーに送信されません。端末内だけで動きます。

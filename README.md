# 歌詞ハイ！タイマー

スマホで動くPWAです。MP3を選んで、歌詞を貼って、フレーズが来た瞬間に「ハイ！」を押すと、LRC / SRT / CSVを出力できます。

## 使い方
1. このフォルダをHTTPSで公開します。Vercel / Netlify / GitHub PagesなどでOKです。
2. スマホでURLを開きます。
3. ブラウザのメニューから「ホーム画面に追加」します。
4. 音源を選び、歌詞を貼ります。
5. 曲を再生して、フレーズ頭で「ハイ！」を押します。
6. SRTまたはLRCを書き出します。

## 注意
- PWAとしてインストールするには、基本的にHTTPS配信が必要です。
- 音源ファイルはサーバーにアップロードされず、端末内で再生されます。
- タップは人間の反応ぶん遅れるので、初期値で -120ms 補正しています。

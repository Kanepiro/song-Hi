# 歌詞ハイ！タイマー v5

スマホで動くPWAです。  
MP3を選んで、歌詞を貼って、フレーズが来た瞬間に「ハイ！」を押すと、LRC / SRT / CSVを出力できます。

## v5変更点
- 再生速度ボタンを追加しました：0.5倍 / 0.75倍 / 1.0倍。
- 「次のフレーズ」に加えて「次の次」も表示します。
- 「直前をやり直す」ボタンを追加しました。直前の打刻を消して、その少し前に戻ります。
- 「ハイ！」ボタンの確認音はON/OFFできます。

## 使い方
1. このフォルダをHTTPSで公開します。Vercel / Netlify / GitHub PagesなどでOKです。
2. スマホでURLを開きます。
3. ブラウザのメニューから「ホーム画面に追加」します。
4. 音源を選び、歌詞を貼ります。
5. 0.5倍または0.75倍で曲を再生して、フレーズ頭で「ハイ！」を押します。
6. SRTまたはLRCを書き出します。

## 注意
- PWAとしてインストールするには、基本的にHTTPS配信が必要です。
- 音源ファイルはサーバーにアップロードされず、端末内で再生されます。
- スロー再生しても、記録される時刻は元曲のタイムラインです。

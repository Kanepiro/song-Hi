# 歌詞ハイ！タイマー v13

曲を再生しながら、歌詞のタイミングで「ハイ！」を押して LRC / SRT を作る PWA です。

## v13 の変更

- オーバーレイ方式を廃止
- 画面を2ページ化
  - メイン：打刻専用画面
  - 設定：音源、歌詞、補正、出力、打刻リスト
- タイトル付近に「設定」ボタンを追加
- 設定画面では同じボタンが「打刻へ」になります
- 旧オーバーレイ画面をメイン画面として表示
- 打刻画面の上下方向を広げ、ボタン間隔を拡大
- ボタン配置は維持
  - 1行目：最初から / ▶ / 消
  - 2行目：ハイ！
- 前 / 今押す / 次 / 次の次 を維持
- 「消」は空のLRCタイムを追加し、次の歌詞には進みません

## 操作

- Space：ハイ！
- Enter：再生 / 停止
- C：消
- R：最初から
- 1 / 2 / 3：再生速度切替

音源は端末内で再生され、サーバーへアップロードされません。

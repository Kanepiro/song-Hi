# 歌詞ハイ！タイマー v14 UI

音源を再生しながら、歌詞が来た瞬間に **ハイ！** を押して LRC / SRT / CSV を作るオフラインPWAです。

## v14の変更点

- スマホ実機で **ハイ！ボタンが画面下に張り付きすぎる問題** を修正しました
- 打刻画面を上寄せにして、ボタンが下端・ナビゲーションバーに近づきすぎないようにしました
- ハイ！ボタンの下に余白を確保しました
- 前 / 今押す / 次 / 次の次 のカンペは残しています
- 表示中プレビューも残しています
- ボタン配置は維持しています
  - 1行目：**最初から / ▶ / 消**
  - 2行目：**ハイ！**
- モバイル表示を少しコンパクト化し、実機で全体が収まりやすいようにしました
- サービスワーカーのキャッシュ名を v14 に更新しました

## 引き続き入っている機能

- **設定 / 打刻へ** の2ページ切替
- 小さい **消** ボタン
  - 押した時刻に空のLRC行を追加します
  - 例：`[01:56.00]`
  - 間奏で歌詞を消したい時に使います
  - **消** を押しても、次の歌詞には進みません
- 「表示中」プレビュー
  - 現在の再生位置で実際に出ている歌詞を確認できます
  - 空行タイム後は「消えています」と表示します
- 戻す系ボタンなし
  - ミスした場合は最初から録り直す運用を想定

## キーボード操作

- Space：ハイ！
- Enter：再生 / 停止
- C：消
- R：最初から
- 1 / 2 / 3：0.5倍 / 0.75倍 / 1.0倍

## 使い方

1. 設定画面で音源を読み込む
2. 歌詞を1行ずつ貼る
3. 「歌詞をセット」
4. 「打刻へ」で打刻画面に戻る
5. 曲を再生
6. 歌い出しに合わせて **ハイ！**
7. 間奏などで字幕を消したいところは **消**
8. 設定画面で LRC 出力

音源や歌詞はサーバーに送信されません。端末内だけで動きます。

# 歌詞ハイ！タイマー v3

スマホで動くPWAです。  
MP3を選んで、歌詞を貼って、フレーズが来た瞬間に「ハイ！」を押すと、LRC / SRT / CSVを出力できます。

## v3変更点
- ローマ字歌詞入力欄を削除しました。
- 歌詞入力欄を1つにしました。
- 「ハイ！」ボタンの上に、現在のフレーズと次のフレーズを表示します。
- スマホでカードが別々にスクロールして見える問題を回避するため、固定表示を解除しています。

## 使い方
1. このフォルダをHTTPSで公開します。Vercel / Netlify / GitHub PagesなどでOKです。
2. スマホでURLを開きます。
3. ブラウザのメニューから「ホーム画面に追加」します。
4. 音源を選び、歌詞を貼ります。
5. 曲を再生して、フレーズ頭で「ハイ！」を押します。
6. SRTまたはLRCを書き出します。


## v4変更点
- 「ハイ！」ボタンを押した時に短い確認音が鳴るようにしました。
- 音はチェックボックスでON/OFFできます。
